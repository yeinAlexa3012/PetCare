## Descripción

PetCare es una aplicación móvil desarrollada con React Native que permite gestionar y consultar información de mascotas.
Incluye navegación entre pantallas, visualización de datos y registro de mascotas.
Además, utiliza Hooks para el manejo dinámico del estado.

---

## Instrucciones de instalación y ejecución

### Clonar el repositorio

```bash
git clone https://github.com/yeinAlexa3012/PetCare.git
cd PetCare
```

---

### Instalar dependencias

```bash
npm install
```

---

### Instalar dependencias adicionales 

```bash
npm install @react-navigation/native
npm install @react-navigation/native-stack
npm install @react-navigation/bottom-tabs
npm install react-native-screens
npm install react-native-safe-area-context
npm install react-native-vector-icons
```

---

### Configuración del Android SDK, solo si presenta errores de conexión

Crear un archivo llamado `local.properties` dentro de la carpeta `android` del proyecto y agregar la ruta 
del Android SDK

---

### Ejecutar la aplicación

1. Activar el modo desarrollador en el dispositivo móvil
2. Activar la depuración USB
3. Conectar el dispositivo al computador

Luego ejecutar:

```bash
npx react-native start
```
Esto para inicializar metro.

En otra terminal:

```bash
npx react-native run-android
```
Esto realizara la respectiva ejecución e instalación de la aplicación.
